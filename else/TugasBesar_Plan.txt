5                                                          1                          SmartBank (Core)  SmartBank adalah inti sistem ekonomi yang menjadi satu-satunya otoritas untuk seluruh transaksi keuangan. Semua pembayaran dari aplikasi lain harus diproses melalui SmartBank. Sistem ini mengelola saldo, transfer, pembayaran, pajak/fee, pinjaman, dan pencatatan ledger sebagai single source of truth.    Regulator & payment processor; pengendali money flow dan money supply.  Validasi saldo; proses debit/kredit; distribusi dana; potong pajak/fee; kelola pinjaman; catat ledger; expose API pembayaran dan query saldo/riwayat.  payment request (from_app, from_user, to_user/service, amount, metadata), loan request  status transaksi, bukti pembayaran, update saldo, ledger entry  Menerima request dari Marketplace, POS, SupplierHub, LogistiKita via API Gateway; menyediakan data ke UMKM Insight (read-only).  Tidak mengelola katalog produk, stok, atau pengiriman; hanya keuangan dan ledger. Semua perubahan saldo hanya boleh terjadi di sini.  POST /payment untuk checkout marketplace; GET /balance untuk cek saldo user.
6                                                          2                   Marketplace (PasarKita)                                                                                              Marketplace menyediakan kanal jual beli produk UMKM. Aplikasi ini mengelola katalog, browsing, dan checkout. Saat checkout, sistem menghitung total + fee marketplace dan mengirim payment request ke SmartBank.           Demand generator (B2C); memicu arus uang dari konsumen ke UMKM.
